  <action>
**src/render/output.ts** — JS-side text renderer adapted from Ink's Output class:
- Maintains a `Map<number, Map<number, string>>` (row -> col -> char) representing a character grid
- `write(x: number, y: number, text: string, options?: { transformers?: Array<(s: string) => string> })`: places each character at (x+i, y), applying transformers. Use string-width for display width awareness.
- `get(width: number, height: number): string`: build output string by iterating rows 0..height-1, for each row iterate cols 0..width-1, join rows with '\n', trim trailing empty rows.
- Handle ANSI escape codes in text — they occupy zero display width but must be preserved in output.

**src/render/render-to-string.ts:**
- `renderToString(node: ReactNode, options?: { columns?: number; rows?: number }): string`
- Creates a temporary ink-root DOM node
- Creates a reconciler container, calls updateContainer synchronously
- Calls calculateLayout with provided or default dimensions (80x24)
- Walks the laid-out DOM tree, painting to an Output instance:
  - For text nodes: write text content at computed (x, y) with style transformers (color, bold, etc.)
  - For box nodes: paint borders and background if specified, then recurse children
- Returns `output.get(width, height)`
- Cleans up: updateContainer(null) to unmount

**src/render/measure-element.ts:**
- `measureElement(ref: RefObject<DOMElement>): { width: number; height: number }`
- Guard: if !ref.current or !ref.current.yogaNode, return { width: 0, height: 0 }
- Return { width: yogaNode.getComputedWidth(), height: yogaNode.getComputedHeight() }

**Tests:**
- renderToString: verify simple text, nested boxes, styled text produce expected strings
- measureElement: verify returns correct dimensions from mock yogaNode, returns zeros for null ref

**src/components/Static.tsx:**
- `Static<T>({ items, children, style }: { items: T[]; children: (item: T, index: number) => ReactNode; style?: Styles })`
- Uses a ref to track `previousCount` of rendered items
- On each render, only renders items from `previousCount` to `items.length`
- The component's DOM node must have `internal_static = true` set — use the reconciler's node creation. The Static component wraps children in a Box and the render() pipeline detects `internal_static` nodes, renders them to string via Output, writes to stdout directly, then removes them from the tree before sending to bridge.
- Mark the root element with a special prop or use `internal_static` flag on the DOMElement (already typed in dom/types.ts)

**src/render/patch-console.ts:**
- `patchConsole(write: (text: string) => void): () => void`
- Saves originals: `const orig = { log: console.log, warn: console.warn, error: console.error }`
- Replaces each with: `console.log = (...args) => write(util.format(...args) + '\n')`
- Returns restore function: `() => { console.log = orig.log; ... }`
- Import `util` from 'node:util'

**src/render/index.ts:**
- Re-export: render, renderToString, measureElement, patchConsole, Instance, RenderOptions

**Tests:**
- Static: verify internal_static flag set, only new items rendered, children function called correctly
- patchConsole: verify console.log/warn/error intercepted, restore function works
