    - Instance.waitUntilExit() resolves after unmount()
    - Instance.clear() writes ANSI clear to stdout
    - Resize event on stdout triggers re-layout and bridge.sendResize
    - render() wraps element in context providers (AppContext, StdinContext, FocusContext)
    - exitOnCtrlC: true (default) adds stdin listener for ctrl+c that calls exit
  </behavior>
  <action>
Implement `render(node: ReactNode, options?: RenderOptions): Instance` following the pattern from RESEARCH.md:

1. **Setup streams:** Default stdout/stdin/stderr to process streams
2. **Create root DOM node:** `createNode('ink-root')` from src/dom/dom.ts
3. **Create bridge:** `new IpcRendererBridge({ maxFps: options.maxFps ?? 30 })`
4. **Wire onComputeLayout:** `rootNode.onComputeLayout = () => calculateLayout(rootNode, stdout.columns, stdout.rows)`
5. **Wire onRender:** `rootNode.onRender = () => { const tree = serializeTree(rootNode); bridge.enqueueRender(tree); }`
6. **Handle first-frame pitfall (Pitfall 4 from research):** After bridge.waitForReady() resolves, call rootNode.onRender() to push any frame dropped during startup
7. **Resize wiring (LYOT-07, RUST-10):** Listen to stdout 'resize', call onComputeLayout + onRender, then bridge.sendResize(stdout.columns, stdout.rows). Send resize FIRST (immediate), then enqueue render.
8. **Create context values:**
   - AppContext: `{ exit: (err?) => { unmount internally; err ? reject(exitPromise) : resolve(exitPromise) } }`
   - StdinContext: `{ stdin, isRawModeSupported: typeof stdin.setRawMode === 'function', setRawMode: refCountedWrapper, internal_exitOnCtrlC: options.exitOnCtrlC ?? true }`
   - FocusContext: Create FocusManager state (ordered ids array, activeId, isFocusEnabled) with add/remove/next/previous/focus/enable/disable methods
9. **Wrap element in providers:** `<AppContext.Provider><StdinContext.Provider><FocusContext.Provider>{node}</FocusContext.Provider></StdinContext.Provider></AppContext.Provider>`
10. **Create reconciler container and initial render:** `reconciler.createContainer(rootNode, ...)` then `reconciler.updateContainer(wrappedNode, container, null, null)`
11. **exitOnCtrlC:** If enabled, add stdin 'data' listener that checks for \x03 (ctrl+c) and calls exit()
12. **Instance methods:**
    - `rerender(nextNode)`: wrap in providers, `reconciler.updateContainer(wrappedNode, container, null, null)`
    - `unmount()`: remove resize listener, remove ctrl+c listener, `reconciler.updateContainer(null, container, null, cb)`, in callback: `bridge.shutdown().then(() => resolveExitPromise())`
    - `waitUntilExit()`: return the stored exit promise
    - `clear()`: write ansi-escapes clearTerminal to stdout

**Tests:** Use mocked IpcRendererBridge (vi.mock src/bridge/ipc-child.ts). Verify:
