# Plan 4: DOM Module — Node Creation & Tree Mutation

## Objective
Create the DOM module that manages the virtual node tree. This includes node creation (all 5 Ink node types), tree mutation (appendChild, removeChild, insertBefore), and Yoga node lifecycle (creation, mirroring, measure function attachment, memory cleanup).

## Context
- Adapted from Ink v6.8.0 `src/dom.ts` (~230 lines)
- Every `DOMElement` gets a paired `YogaNode` via `Yoga.Node.create()` — this is the RECON-05 requirement (Yoga tree mirrors DOM tree)
- Text nodes (`ink-text`) get a measure function attached via `yogaNode.setMeasureFunc()` — this is how Yoga knows text dimensions
- The measure function calls `measureText()` and `wrapText()` from the layout module (Plan 2)
- `squashTextNodes()` flattens a text node's child tree into a single string before measuring
- Yoga memory management is critical: every `removeChild` must call `yogaNode.freeRecursive()` after `unsetMeasureFunc()` to prevent WASM memory leaks
- The `onComputeLayout` callback is set by the caller (reconciler/test) and triggers `yogaNode.calculateLayout()` — this is how LYOT-01 layout computation happens
- Types are imported from `./types.js` (Plan 2)

### 1b. createNode() function

This function creates DOM nodes with their paired Yoga nodes. It must handle all 5 node types:

```typescript
export function createNode(nodeName: string): DOMElement {
  const node: DOMElement = {
    nodeName,
    attributes: {},
    childNodes: [],
    parentNode: undefined,
    yogaNode: nodeName === 'ink-virtual-text' ? undefined : Yoga.Node.create(),
    internal_transform: undefined,
    style: {},
    internal_layoutListeners: undefined,
  };

  if (nodeName === 'ink-text') {
    // Text nodes need a measure function so Yoga can query text dimensions
    node.yogaNode?.setMeasureFunc(measureTextNode.bind(null, node));
  }

  return node;
}
```

The 5 node types and their Yoga behavior:
| Node Type | Gets YogaNode? | Gets MeasureFunc? |
|-----------|---------------|-------------------|
| `ink-root` | Yes | No |
| `ink-box` | Yes | No |
| `ink-text` | Yes | Yes — measures text content |
| `ink-virtual-text` | No — layout handled by parent ink-text | No |
| `#text` | No — leaf text content | No |

### 1d. measureTextNode() — Yoga Measure Callback

This is the function bound to `ink-text` nodes. It's called by Yoga during `calculateLayout()`:

```typescript
function measureTextNode(
  node: DOMElement,
  width: number,
): {width: number; height: number} {
  const text = squashTextNodes(node);

  // If no text content, return zero dimensions
  if (text.length === 0) {
    return {width: 0, height: 0};
  }

  // Get the max width considering padding and border
  const maxWidth = getMaxWidth(node.yogaNode!);

  // Wrap/truncate text based on textWrap style
  const wrappedText = wrapText(text, maxWidth, node.style.textWrap ?? 'wrap');

  return measureText(wrappedText);
}
```

**Important:** The measure function signature for yoga-layout 3.2.1 is `(width, widthMode, height, heightMode) => {width, height}`. Ink's code binds `node` as the first arg, so the actual function receives `(width, widthMode, height, heightMode)` but only uses `width`. The extra parameters are harmlessly ignored. Copy Ink's exact implementation.

### 1e. appendChild() / removeChild() / insertBefore()

These functions manage both the DOM tree and the Yoga tree in parallel:

```typescript
export function appendChild(node: DOMElement, child: DOMNode): void {
  if (child.parentNode) {
    removeChild(child.parentNode, child);
  }

  child.parentNode = node;
  node.childNodes.push(child);

  // Mirror in Yoga tree
  if (child.nodeName !== '#text' && child.nodeName !== 'ink-virtual-text') {
    const element = child as DOMElement;
    if (element.yogaNode && node.yogaNode) {
      node.yogaNode.insertChild(
        element.yogaNode,
        node.yogaNode.getChildCount(),
      );
    }
  }
}
```

```typescript
export function removeChild(node: DOMElement, child: DOMNode): void {
  child.parentNode = undefined;
  const index = node.childNodes.indexOf(child);
  if (index >= 0) {
    node.childNodes.splice(index, 1);
  }

  // Clean up Yoga node (CRITICAL for memory management)
  if (child.nodeName !== '#text' && child.nodeName !== 'ink-virtual-text') {
    const element = child as DOMElement;
    if (element.yogaNode && node.yogaNode) {
      node.yogaNode.removeChild(element.yogaNode);
      cleanupYogaNode(element.yogaNode);
    }
  }
}
```

```typescript
export function insertBefore(
  node: DOMElement,
  child: DOMNode,
  beforeChild: DOMNode,
): void {
  if (child.parentNode) {
    removeChild(child.parentNode, child);
  }

  child.parentNode = node;
  const beforeIndex = node.childNodes.indexOf(beforeChild);
  if (beforeIndex >= 0) {
    node.childNodes.splice(beforeIndex, 0, child);
  } else {
    node.childNodes.push(child);
  }

  // Mirror in Yoga tree at correct position
  if (child.nodeName !== '#text' && child.nodeName !== 'ink-virtual-text') {
    const element = child as DOMElement;
    if (element.yogaNode && node.yogaNode) {
      // Count only Yoga-enabled siblings before this position
      const yogaIndex = getYogaChildIndex(node, child);
      node.yogaNode.insertChild(element.yogaNode, yogaIndex);
    }
  }
}
```

### 1f. cleanupYogaNode() — WASM Memory Cleanup

```typescript
function cleanupYogaNode(node?: YogaNode): void {
  node?.unsetMeasureFunc();
  node?.freeRecursive();
}
```

This MUST be called in `removeChild` and `removeChildFromContainer`. Without it, every removed node leaks WASM memory.
